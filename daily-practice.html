five-minute return

The daily sequence.

  1. Arrive

    Put one hand on the body. Feel the ground. Stop optimizing the practice.

  2. Read

    Name the strongest signal in the body. No interpretation yet.

  3. Separate

    Name the mind program if it is running. “A part of me wants to…”

  4. Ask

    What is the deeper direction here, if I do not betray the body?

  5. Act

    Choose one move that can be done today. Small enough to be real.

one sentence

“What is the honest next move?”

If you leave with only one question, use this one.
